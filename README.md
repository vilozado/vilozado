### [Litlas](https://github.com/vilozado/Litlas)
Independently engineered full-stack application for finding world-wide book recommendations through interative map-based experience.
<ul>
  <li>Built with React, Node.js, Express, MongoDB, React-Leaflet and the Google Books API</li>
  <li>Architected scalable backend using MVC pattern and implemented RESTful APIs with CRUD operations</li>
  <li>Integrated Google Books API and reduced response times via in-memory caching</li>
  <li>Designed an interactive map-based UI to visualize global literature and user reading activity</li>
</ul>
